# Student Performance Analysis Using Data Visualization

## Objective
Analyze student academic performance and attendance patterns using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed
- Data Loading
- Average Score Calculation
- Mean, Median, Standard Deviation
- Attendance Analysis

## Visualizations
1. Bar Chart
2. Scatter Plot
3. Correlation Heatmap
4. Box Plot

## Key Findings
- Students with attendance above 85% achieved higher average scores.
- Attendance shows a strong positive correlation with academic performance.
- Science had the highest average score in the sample dataset.
